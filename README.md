## Project Overview

*Blood Connect* is a simple, 4-page static website designed to connect potential blood donors with individuals or facilities urgently needing specific blood types. This project was developed as a submission for a first-year college Web Development course, focusing strictly on foundational web technologies.

*Goal:* To demonstrate proficiency in structuring a multi-page site using core HTML, CSS, and basic JavaScript.

## âœ¨ Site Features

The website contains the required four distinct pages, each with its own purpose:

1.  **Home (index.html):**
    * Introduction to the mission.
    * Call-to-action button linked to a basic JavaScript alert function (displayAlert()).
2.  **Search (search.html):**
    * Allows users to select a blood type and enter a location.
    * Uses JavaScript (handleSearch()) to simulate finding results and provide conditional messages (e.g., highlighting the need for O-).
3.  **Register (register.html):**
    * A form for users to register as a Donor or Receiver.
    * Uses JavaScript (handleRegistration()) for basic client-side validation (10-digit phone number check) and confirmation alerts.
4.  **About (about.html):**
    * Details the mission, the importance of blood donation, and contact information.
